# particl.page

## Shortcodes reference

Links:

```
[xxxxxxx]({{< ref "xxxx/xxxx.md" >}})
```

Image:

```
{{< figure
  title=""
  caption="Description"
  src="/images/image.png"
  link="/images/image.png"
  attrlink=""
  attr=""
>}}
```

Video:

```
{{< video
  class=""
  poster=""
  mp4="/video.mp4"
  webm="/video.webm"
  ogg="/video.ogg"
>}}
```
